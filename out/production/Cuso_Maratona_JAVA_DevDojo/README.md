# ☕ Maratona Java — DevDojo

A repository containing all my **notes and exercises** from the **Maratona Java** course by the [DevDojo](https://www.youtube.com/@DevDojoBrasil) YouTube channel — one of the most complete free Java courses available in Brazil.

> ⚠️ This repository is constantly being updated — I'm still working through the course!

---

## 🎯 Goal

Store and organize every lesson and exercise from the Maratona Java course, documenting my learning journey through Java from the very basics all the way to more advanced topics as I progress.
